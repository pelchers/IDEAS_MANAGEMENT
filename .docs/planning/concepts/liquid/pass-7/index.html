Description

Wavz.fm is a next-generation music streaming platform designed specifically for independent musicians. The platform provides artist-first tools including real-time analytics, fan engagement features, and a fair revenue-sharing model. The frontend is built with Next.js App Router and styled with Tailwind CSS, while the backend leverages Convex for real-time data synchronization. Authentication is handled through Clerk, and payments are processed via Stripe Connect.

schema.ts TypeScript 
import { defineSchema, defineTable } from "convex/server";
import { v } from "convex/values";

export default defineSchema({
  users: defineTable({
    email: v.string(),
    name: v.string(),
    avatarUrl: v.optional(v.string()),
    role: v.union(
      v.literal("admin"),
      v.literal("member")
    ),
  }),
  projects: defineTable({
    name: v.string(),
    slug: v.string(),
    ownerId: v.id("users"),
    status: v.string(),
  }),
});
