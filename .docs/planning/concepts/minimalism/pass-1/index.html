Design System

This document outlines the design system specifications for the Mobile App Redesign project. It covers color tokens, typography scale, spacing grid, and component patterns used across all screens.

Color Tokens

The primary palette uses a monochromatic blue scale with warm neutrals. All tokens are defined as CSS custom properties for easy theme switching.

--color-primary: #0066FF; --color-surface: #FAFAFA; --color-text: #333333; --color-muted: #999999;

Typography Scale

We use a modular scale (1.25 ratio) starting from a 16px base. Headings use Inter SemiBold (600), body uses Inter Regular (400).

Spacing

All spacing follows an 8px grid. Components use multiples: 8, 16, 24, 32, 48, 64px. No odd values.
