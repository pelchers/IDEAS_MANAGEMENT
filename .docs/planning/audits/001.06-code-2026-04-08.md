## Recommendations
1. Extract whiteboard and schema pages into multiple focused modules: separate type definitions, canvas rendering engine, toolbar component, modals, and AI integration files.
2. Create a centralised Zod request validation wrapper or middleware for API routes, and apply it to all 26 unvalidated routes.
3. Replace `window.__imSelectProject` with a lightweight Zustand store or React Context for selected project state.
