# Task Decomposition — Usage Guide

## Quick Start
```
"decompose this feature into implementation tasks"
```

## Prioritization Methods

### MoSCoW
- **Must:** Essential for launch
- **Should:** Important but not critical
- **Could:** Nice to have
- **Won't:** Explicitly deferred

### Eisenhower Matrix
- **Do First:** Urgent + Important
- **Schedule:** Important, not urgent
- **Delegate:** Urgent, not important
- **Eliminate:** Neither

## Output
Numbered task list with dependencies, complexity scores, and priority labels.
