# Component Creator System

## Overview
Automates the full pipeline for creating new agents, skills, hooks, commands, and system docs. Ensures no gaps in the creation process.

## Components
| Component | Path |
|---|---|
| **Agent** | `.claude/agents/component-creator-agent/AGENT.md` |
| **Skill** | `.claude/skills/component-creation-pipeline/SKILL.md` |
| **Command** | `.claude/commands/create-component.md` |

## Usage
```
/create-component "description"
/create-component --complete <name>
/create-component --audit
```
