# Game Development System

## Overview
Comprehensive game development agents and skills covering ECS, graphics, animation, multiplayer, physics, audio, AI/pathfinding, UI/UX, and DevOps. Includes both a general game-dev agent and a Godot-specific agent.

## Components
| Component | Path |
|---|---|
| **Game Dev Agent** | `.claude/agents/game-development-agent/AGENT.md` |
| **Game Dev Skill** | `.claude/skills/game-development-agent/SKILL.md` |
| **Godot Agent** | `.claude/agents/godot-development-agent/AGENT.md` |
| **Godot Skill** | `.claude/skills/godot-development-agent/SKILL.md` |

## How to Use
```
"help me build a game with ECS architecture"        → game-dev agent
"create a Godot scene with multiplayer networking"   → godot agent
"optimize my game's rendering pipeline"              → game-dev agent
```

## Resources
Both skills include extensive reference files in `resources/` covering: ECS, graphics, animation, audio, AI, physics, multiplayer, procedural generation, performance, and DevOps.
