# Task Decomposition System

## Overview
Breaks down complex assignments and projects into manageable core questions, identifies information gaps, generates research questions, and prioritizes tasks using MoSCoW and Eisenhower Matrix methods.

## Components
| Component | Path |
|---|---|
| **Skill** | `.claude/skills/decomposing-project-tasks/SKILL.md` |

## How to Use
```
"break down this project into tasks"
"decompose the authentication feature into subtasks"
"prioritize these requirements using MoSCoW"
```

## Methods
- **MoSCoW** — Must have, Should have, Could have, Won't have
- **Eisenhower Matrix** — Urgent+Important, Important, Urgent, Neither
- **Core Questions** — What, Why, How, When, Who, Dependencies
