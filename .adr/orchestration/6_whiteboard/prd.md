# PRD — 6_whiteboard

## Summary
Build the whiteboard as a faithful pass-1 conversion with HTML5 canvas, drawing tools, sticky notes, and Rough.js decorations.

## Goals
- Canvas with grid background matching pass-1 (30px grid)
