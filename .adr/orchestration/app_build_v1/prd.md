## Session Summary
Deliver a production-ready, deployable web + desktop platform for idea management and software planning with:
- Auth + subscriptions + admin override
- AI chat (full page + sidebar) with tool actions that mutate project artifacts under audit control
- Hybrid local/cloud project storage with cloud-canonical sync
