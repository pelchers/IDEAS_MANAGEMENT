# PRD — 8_simple-views

## Summary
Build ideas capture, directory tree, and settings pages as faithful pass-1 conversions. These are grouped because they are lower complexity individually. The directory tree view supports three data source modes (same "trinary" pattern as schema planner): import from local files/pasted text, import from a GitHub repository, or create a custom tree from scratch. All trees are exportable.

## Goals
- Ideas grid with filter chips and priority badges matching pass-1
- Directory tree with expand/collapse and file preview matching pass-1
- Directory tree: trinary data source (Manual, Local, GitHub) + export
- Settings with profile, preferences, integrations, danger zone matching pass-1
- All wired to respective APIs
