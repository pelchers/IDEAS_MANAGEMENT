# PRD — 5_kanban-board

## Summary
Build the kanban board as a faithful pass-1 conversion with SortableJS drag-and-drop, colorized column headers, and card management, wired to the artifact API.

## Goals
- Kanban visually identical to pass-1 (4 columns, colored headers, card tags)
- SortableJS drag-and-drop with same animation timing (200ms)
- Card CRUD and column management
- Persistent storage via artifact API

## Key Reference
- Pass-1 kanban: 4 columns with colored headers, 12 cards, SortableJS drag-drop, tag colors
