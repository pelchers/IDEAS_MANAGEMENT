# Automated Job Application System

**Idea #10** — captured 2026-05-12.

## One-liner
A flag-driven, multi-platform job application engine that runs **tiered cycles** — exact-match jobs first, partial-match next, corollary-area "bulk similar" last — with per-cycle count caps, AI-tailored resumes against a user-managed master resume, and a review UI that combines code automation (browser/ATS form-fill) with AI automation (cover letters, screening answers).
