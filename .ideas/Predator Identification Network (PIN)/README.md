# Predator Identification Network (PIN)

**Idea #9** — captured 2026-05-11.

## One-liner
A collaborative platform — branded **PIN (Predator Identification Network)** — where independent investigators ("poachers"), journalists, experts, legal representatives, and the public document and identify predators, with an AI-gated cross-confirmation step before any predator is publicly listed, and a premium-content subscription model that distributes revenue equitably to participating poachers.
