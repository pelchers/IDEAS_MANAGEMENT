# Overview

`Mobile SSH Repo Terminal Companion` is a React Native + Expo mobile application that acts as a remote control and coding companion for a user's PC. The app connects over SSH, establishes a terminal-capable session, and lets the user work with coding assistants such as Codex or Claude Code from a phone while still operating against the PC's local environment.

The product is not just an SSH client. It combines a terminal surface, repository discovery, and remote filesystem access into one workflow-oriented interface. It should help a user find a repo, connect to the host machine, inspect files, and interact with an agentic coding workflow without switching between multiple apps.

The defining product bet is that mobile can be a legitimate remote coding control plane when the session quality, repo discovery, and file operations are intentionally designed instead of bolted onto a generic SSH shell.
