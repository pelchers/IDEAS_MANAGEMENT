# User Stories

## Connection

- As a user, I want to save a PC connection profile so I can reconnect quickly.
- As a user, I want key-based SSH authentication so I do not have to re-enter a password constantly.
- As a user, I want recent hosts and sessions so I can resume my last workflow.

## Terminal

- As a user, I want an interactive terminal from my phone so I can run Codex or Claude Code remotely.
- As a user, I want terminal scrollback and stable rendering so long outputs remain usable.
- As a user, I want copy/paste and keyboard shortcuts where possible so mobile interaction is less painful.

## Repository Discovery

- As a user, I want the app to find repositories on my PC automatically so I do not have to remember every path.
- As a user, I want list, table, and grid views so I can switch between dense and visual layouts.
- As a user, I want search and sorting so I can find the right repository fast.

## Filesystem

- As a user, I want to browse my PC filesystem from `C:\` so I can inspect files remotely.
- As a user, I want repo entries to open their real paths so I can move from discovery into action.
- As a user, I want destructive operations to be constrained so I do not accidentally damage the host machine.
